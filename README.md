# Titanic OOP Analysis Project

## Overview

This project analyzes the Titanic dataset using Object-Oriented Programming principles in Python.

## Features

- Data Loading from URL
- Data Cleaning
- Survival Statistics
- Outlier Detection using DSA
- Age Distribution Histogram
- Survival Distribution Visualization
- OOP Concepts:
  - Encapsulation
  - Inheritance
  - Abstraction
  - Polymorphism

## Technologies

- Python
- Pandas
- Matplotlib

## Dataset

Titanic Dataset from Data Science Dojo

## Results

- Total Passengers: 891
- Total Survivors: 342
- Survival Rate: 38.38%
- Average Age: 29.70

- Age Histogram
- Survival Bar Chart
- Console Output
- Highest Fare: 512.33
